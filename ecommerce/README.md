# Ecommerce DRF API

A backend Ecommerce API project developed using Python, Django, and Django REST Framework (DRF).

## Features

* Product Management API
* User Authentication
* RESTful APIs
* Cart Functionality
* Database Integration
* Image Upload Support

## Technologies Used

* Python
* Django
* Django REST Framework (DRF)
* MySQL
* HTML
* Git & GitHub

## Project Purpose

This project was developed for learning backend development, REST APIs, and Django REST Framework concepts.
